# 🌫️ MP4-NOICE

動画を「完全なノイズ（砂嵐）」へ変換し、動いているものだけをノイズの揺らぎとして抽出する、プライバシー保護＆アーティスティックなビデオプロセッサ。

## 🛠️ プロジェクト詳細
- **連絡先**: [pqlpp@proton.me](mailto:pqlpp@proton.me)
- **GitHub**: [https://github.com/qawsedrftgyhujikolpa/mp4-noice](https://github.com/qawsedrftgyhujikolpa/mp4-noice)

## 🎥 特徴
- **真・ノイズ迷彩**: 静止している物体は背景の固定ノイズと同化して完全に見えなくなります。
- **動体抽出**: 動いている領域だけが、リアルタイムに変化する動的ノイズとして浮かび上がります。

## 🚀 使い方
1. `run_app.bat` を実行してください。
2. 自動的にブラウザが開き、[http://127.0.0.1:8000](http://127.0.0.1:8000) で操作可能になります。
